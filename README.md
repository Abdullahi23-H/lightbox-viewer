Lightbox Gallery

A simple interactive image gallery that displays full-size images in a lightbox when a thumbnail is clicked.


Features

Click a thumbnail to view the full-size image in a popup overlay.

Close the lightbox by clicking the close button or clicking outside the image.

Responsive and easy-to-use layout.
